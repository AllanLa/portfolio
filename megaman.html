Simple Megaman Game

In my second computer science class at Bucknell University, we had a project that was meant to string together our knowledge of data structures at the time. At that point in time, we learned arrays, linked lists, stacks, and queues. This project is meant to utilize all these data structures and good object design principles. Depending on what data structures were used and how the code is designed, the project could be extremely simple or complicated. For this project, I decided to make a simple Megaman game. My inspiration came off of the classic Megaman Battle Network gameboy games! It uses the same charecters and maps. The objective of the game is to collect battle chips to gain attack. Megaman then has travel throughout the game to defeat enemies listed in the objective area. It was a great learning experience for me to understand good object oriented design and how key choices in data structures can change a lot of things.
