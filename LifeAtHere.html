LifeAtHere, The Complete Full Stack Development

Senior Design, the classic final graduation class for engineers. For me, it was a class to prove oneself. What can I do to show the four hard years spent at Bucknell was worth it? I wanted to do something I would be proud of, something new, something I never attempted before. Senior Design was the perfect stage for this. My team was assigned a full stack development projec. We didn't have too much experience with yet. We all had different skills. Learning our strengths and weaknesses were important for a successful project. We all researched throughly what would be needed to accomplish our project and with the guide of Alan Marchiori, we would eventually start the design process.

Essentially, our project needed a web application, a server and a database. We wanted a server to process requests from the frontend to the backend and a server that would also communicate to our database to pull information to send it back to the frontend. We decided to use MongoDB as our database and Python Flask as our server. We would code the frontend using ReactJS for scalability and cross-browser compatibility. We choosed Python Flash because it was easy to get it up and running, although we would later run into performance issues due to it's limitations. MongoDB since it was easy to learn and use. Our team learned so much from this project. We started with no clue on how to approach this project and now we have a fully functioning product that our Bucknell Campus actually uses.
